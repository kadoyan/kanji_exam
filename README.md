# 漢字テスト作成　Svelte + TS + Vite

JSONファイルに漢字と読みを設定し、それをもとに一覧を作成。  
一覧でチェックしたものを漢字テストとして表示します。  
A4横で印刷すると、だいたいちょうどいい漢字にプリントアウトできると思います。

フォントはローカル依存なので、必要であればインストールしてください。

JSONは次のような書式になっています。

```
[
	{
		"kanji": "ここに{問題文}を書く"
		"yomi": "もんだいぶん"
	},
	{
		"kanji": "中括弧が四角に{差し代わる}",
		"yomi": "さしかわる"
	}
]
```
